# Tarea #1 Yordi Elizondo Montes

## 6. Detalle Técnico

### 6.1. Herramientas utilizadas para el desarrollo

- Visual Studio (Se recomienda 2019 o superior)
- .NET Framework 4.8
- ADO.NET
- SQL Server
- Microsoft SQL Server Management Studio 18
- jQuery 3.7.0

### 6.2. Herramientas requeridas para la instalación

- .NET Framework 4.8 (debe estar instalado en el sistema)
- SQL Server (para la base de datos)
- IIS o IIS Express (para hospedar la aplicación)
- Navegador web moderno (para acceder a la aplicación)
